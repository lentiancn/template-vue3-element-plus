# Vue3 + Element Plus

[![许可证](https://img.shields.io/github/license/lentiancn/template-vue3-element-plus)](LICENSE)

## 介绍

Vue3 + Element Plus 工程模板

## 包含的功能

* Vue (3.5.22)
* TypeScript
* JSX 支持
* Router（单页面应用开发）
* Pinia（状态管理）
* Vitest（单元测试）
* 端到端测试
* ESLint（错误预防）
* Prettier（代码格式化）

## 端到端测试框架

Playwright (https://playwright.dev/)

## 包含的试验特性

Oxlint
